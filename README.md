# snake-ladder-mern

This project is a full-stack Snake and Ladder game using the MERN stack (MongoDB, Express.js, React, Node.js).

## Features
- **Frontend**: React-based 10x10 game board, dice roller, player tokens, game status.
- **Backend**: Node.js + Express REST API for game state management, Socket.io for real-time multiplayer.
- **Database**: MongoDB for storing game sessions, player data, and history.
- **Game Logic**: Snakes, ladders, dice rolls, win conditions.

## Tech Stack
- MongoDB
- Express.js
- React
- Node.js
- Socket.io
- TailwindCSS (if applicable for styling)
